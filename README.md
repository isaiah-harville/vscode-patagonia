# Patagonia

Patagonia is a VS Code theme pack with dark and light variants built around cool blue surfaces, clear syntax contrast, and editor polish that stays out of the way.

It currently includes:

- `Patagonia (Dark)`
- `Patagonia (Light)`
- built-in Vim status bar colors via extension `configurationDefaults`

The theme also includes semantic token colors and tuned workbench colors for brackets, git decorations, minimap highlights, breadcrumbs, tabs, and the integrated terminal.
